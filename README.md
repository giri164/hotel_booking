# GVR Hotel Website

A responsive and user-friendly hotel management website built using **HTML, CSS, and basic JavaScript**. This project provides an elegant interface for hotel room display, gallery showcase, booking registration, and customer contact support.

## 📌 Project Overview

The **GVR Hotel Website** is designed to provide customers with a smooth online hotel browsing and booking experience. It includes multiple sections such as Home, Rooms, Gallery, Booking Registration, and Contact Information.

This project is suitable for beginners learning **Frontend Web Development** and demonstrates the use of:

* HTML5
* CSS3
* JavaScript (Basic Form Handling)
* Responsive Web Design

## ✨ Features

### 🏠 Home Section

* Welcome message for visitors
* Hotel introduction
* Facilities list including:

  * Air-conditioned rooms
  * 24/7 room service
  * Free Wi-Fi
  * Rooftop restaurant
  * Easy city access

### 🛏 Rooms Section

Displays available room types:

* Standard Room
* Deluxe Room
* Suite Room

Each room includes:

* Room image
* Description
* Price per night

### 🖼 Gallery Section

* Attractive hotel image gallery
* Hover animation effects
* Responsive grid layout

### 📝 Booking Registration Form

Allows users to:

* Enter personal details
* Select gender
* Choose booking date
* Upload ID proof
* Select room type
* Submit booking request

### 📞 Contact Section

* Contact form with alert popup
* Hotel address
* Phone number
* Email information
